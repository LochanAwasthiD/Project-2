# HY_DRO Website Project

This is a small beginner website made for a soil meter product called **HY_DRO**.
The project has simple HTML and CSS pages. It is only for practice and learning.

## Folder Structure

- **HTML/**
  - Home.html
  - Products.html
  - ProductDec.html
  - Description.html

- **CSS**
  - Home.css
  - Products.css
  - ProductDec.css
  - Description.css

- **Picture**
  - Images used in the website


## How to Open the Website

1. Open the `HTML` folder
2. Click and open **Home.html**
3. The website will start from the home page
4. Use the buttons to move to other pages


## Features

- Simple navigation between pages
- Sidebar buttons (Home, Products, Description)
- Product details page
- Review/description cards
- Footer with social links


##Purpose

This project is made for:
- practicing basic HTML and CSS
- learning page linking
- improving beginner web design skills

It is **not** connected to any backend or database.
